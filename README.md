# TerrainToolV2
This is my second version of this repository. The goal is to create a tool that can generate a terrain with variablity. It should have multiple capabilities as a asset generator and a functional widget for ease of use.

GitHub URL: https://github.com/Itsjustmeblank/TerrainToolV2.git


Challenge Feature(s):
- Procedural terrain generator tool in Maya using vertex deformation
- Adjustable intensity, radius, and noise
- Mountain and valley generation modes(more attempted and failed)
- Mesh reset system restoring original vertex positions
- Custom Maya shelf integration for quick access
